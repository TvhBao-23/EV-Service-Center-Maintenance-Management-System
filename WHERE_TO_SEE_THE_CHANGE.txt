TRANG 1: Đặt lịch dịch vụ (Customer)
────────────────────────────────────

URL: http://localhost:3000/booking

CÁCH XEM:
   1. Login với tài khoản customer
   2. Click "Đặt lịch" trong menu
   3. Chọn dịch vụ: "Sửa chữa hệ thống sạc"
   4. Cuộn xuống → Thấy box màu xanh blue:
      
      ┌───────────────────────────────────────────────┐
      │ 🎯 Phụ tùng cho dịch vụ này          8 phụ tùng │
      ├───────────────────────────────────────────────┤
      │ CHG-PORT-001 - Cổng Sạc Type 2                │
      │ CHG-CABLE-001 - Dây Sạc Type 2 - 5m           │
      │ CHG-ONBOARD-001 - Bộ Sạc Onboard 11kW         │
      │ ... (5 items nữa)                             │
      │                                               │
      │ 💡 Danh sách này CHỈ hiển thị phụ tùng       │
      │    phù hợp với dịch vụ bạn đã chọn           │
      └───────────────────────────────────────────────┘

KẾT QUẢ: 8 phụ tùng ✅ (thay vì 24!)

THỬ THÊM:
   • Đổi sang "Thay pin lithium-ion" → 3 phụ tùng
   • Đổi sang "Bảo dưỡng định kỳ" → 14 phụ tùng
